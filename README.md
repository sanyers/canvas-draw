# canvas-draw

#### 介绍

canvas 画板，白板

[在线 DEMO](https://sanyers.gitee.io/canvas-draw)
[在线 DEMO github](https://sanyers.github.io/canvas-draw)

#### 软件架构

1. element-ui
2. vue
3. vue cli4

#### 安装教程

1.  git clone
2.  npm install
3.  npm run serve

#### 使用说明

支持画板常用功能

1.  绘制：铅笔、线条、圆、矩形
2.  功能：橡皮擦、撤回、前进、清屏、粗细、颜色
3.  操作：保存、关闭
